ทำไมต้องPython?

   สำหรับผู้ที่เริ่มต้นอาจสงสัยกันจึงได้สรุปข้อดีของPythonดังนี้ครับ

  • สามารถเขียนอุปกรณ์IOTได้เช่นMicroPython

print("A") print("Hello") print('This is String') print('1')

ตัวแปร (Variable)

   สำหรับการเขียนโปรแกรมสิ่งที่ขาดไม่ได้คือตัวแปร ตัวแปรคือการจองทรัพยากรณ์ในคอมพิวเตอร์ เช่น ตัวแปรAคือการจองทรัพยากรชื่อAนั่นเอง

การตั้งชื่อตัวแปรของPythonมีดังนี้

  • ตัวพิมพ์เล็ก/ใหญ่ ถือเป็นคนละตัวกัน
  • ตัวแปรต้องขึ้นต้นต้วยภาษาอังกฤษเท่านั้น
  • ตัวแปรต้องไม่มี จุด,ช่องว่าง,สัญลักษณ์พิเศษ ยกเว้น underscore(_)สามารถใช้ได้

Var = Noneให้ตัวแปรVar เป็นค่าว่างแบบไม่กำหนดชนิดข้อมูล

การเข้าถึงตำแหน่งนข้อมูลชนิดstring

index : ตำแหน่งในstring


Example

start : ตำแหน่งเริ่มต้น

stop : ตำแหน่งสิ้นสุด

step : จำนวนในการข้าม

จะเห็นได้ว่าตำแหน่งสุดท้ายของindexกับการเข้าถึงแบบการกำหนดตำแหน่งจะมีค่าต่างกัน เนื่องจากA[13]หมายถึงเอาตำแน่งที่13 แต่ A[13:14]หมายถึงงเอาตำแหน่ง13และสิ้นสุดก่อนตำแหน่งที่14นั่นเอง

การเขียนโปรแกรมกำหนดเงื่อนไข (Conditional statements)

    สำหรับการเขียนโปรแกรมสิ่งที่ขาดไม่ได้เลยคือ การเขียนโปรแกรมกำหนดเงื่อนไข (Conditional statements) โดยที่ใช้ใช้booleanในการกำหนดทางเลือก

คำสั่ง if

คำสั่งนี้โปรแกรมจะเช็คว่าจริงหรือไม่ หากจริงจะทำตามคำสั่งในif ถ้าไม่ก็จะข้ามไป โดยจะมีcolon(:) จบคำสั่งเสมอ

if condition :
   statement 1
   statement 2
   ...
   statement n

Example

X = 40; Y = 30
if X > Y :
   print("X is greater than Y")
print("Finish!!")
 output: X is greater than Y
output: Finish!!

เนื่องจาก X < Y เป็นจริงทำให้ดำเนินคำสั่ง print("X is greater than Y") หลังจากออกนอกคำสั่ง จึงทำคำสั่งprint("Finish!!") ต่อ

if condition :
   statement 1
else :
   statement 2

Example

X = 40; Y = 30
if X > Y :
   print("X is greater than Y")
else:
   print("Y is greater than X")
 output: X is greater than Y

   สำหรับคำสั่ง for นั้นจะแตกต่างตั่งที่มีจำนวนรอบมากำหนดการแสดงผลโดยนับจำนวนครั้งจนถึงจุดสิ้นสุดของการวนลูป เช่น

 output: 0

Example

c = ‘Meow’ #กำหนดตัวแปร s เก็บข้อมูลสตริง
